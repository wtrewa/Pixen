# Premium Wedding Photography Marketplace — Design System + Product Skills

## Product Vision

Build a premium wedding photography marketplace that feels like:

* Apple-level premium UI
* Airbnb-level trust-driven booking
* Wedding-focused emotional luxury platform

This is NOT a normal marketplace.
This is a memory-selling platform.

Users should feel:
"This platform handles life’s most important memories."

---

# Core Product Type

## Two-Sided Marketplace

### Side A — Customers

People looking for wedding photographers and videographers.

### Side B — Vendors

Photographers, cinematographers, studios, freelancers.

### Side C — Admin

Platform operations, approvals, fraud prevention.

# Spacing Rules

Luxury UI = whitespace

Use:

* large padding
* clean spacing
* breathing room

Avoid:

* crowded UI
* tight cards
* too much information above the fold

Rule:
Less clutter = more premium

---

# Radius Rules

Use:
18px to 24px border radius

For:

* cards
* buttons
* modals
* inputs
* portfolio previews

This creates iOS premium feel.

---

# Shadow Rules

Use:
Soft subtle shadows only

Avoid:

* hard borders
* heavy black shadows
* old-style box shadows

Goal:
Luxury softness
